# Next.js Resume Page

A high-performance single-page portfolio application structured using Next.js (App Router) architectural patterns and styled exclusively with utility-first Tailwind CSS classes.

## Features Implemented
* **Semantic HTML Elements:** Built using clean landmark structures (`<main>`, `<header>`, `<section>`) along with explicit structural layout indicators to ensure high web accessibility profiles.
* **Tailwind Utility Architecture:** Component-free design rendered entirely using embedded Tailwind utility classes via `className` mappings, keeping global style files clean and standardized.
* **Responsive Layouts:** Designed to adapt layout flows across mobile displays and standard high-resolution desktop viewports.
